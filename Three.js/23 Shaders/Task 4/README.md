Set aRandom attribute to geometry that is float created by Math.random() by using setAttribute function and new THREE.BufferAttribute.



Using attribute keyword, fetch that aRandom attribute inside vertex.glsl.



set modelPosition.z to aRandom \* 0.1.



Using varying keyword, pass aRandom as vRandom to fragment.glsl.



Inside fragment.glsl use vRandom to color vertexes based on it.
