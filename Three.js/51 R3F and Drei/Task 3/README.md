Continuation from the previous task.



Import PivotControls, and wrap sphere mesh using it.



Set PivotControls attributes; anchor to 0, 0, 0 (relative to mesh), depthTest false, lineWidth 4, axisColors to colors you want, scale 100, fixed true.



Compared to TransformControls, PivotControls is not a group, so you have to create one PivotControls for each mesh you want it to have.
