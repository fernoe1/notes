Build a react app from scratch.

Create from-scratch folder.

Change directories into it and do npm init -y

Do npm install react@18 react-scripts@5 react-dom@18.2

Inside package.json, create dev and build scripts for running dev server and building the project.

Create public folder and index html with basic code and div with root id.

Create src folder

Import createRoot from react-dom/client.

Using createRoot create root, and render a Hello React message using it.

Do npm run dev.
